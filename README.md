📚 APS_POO

Projeto acadêmico desenvolvido no contexto da disciplina de Programação Orientada a Objetos (POO), com foco na aplicação prática dos conceitos fundamentais da orientação a objetos em um ambiente colaborativo.

Este repositório representa o desenvolvimento de uma solução em grupo, simulando um fluxo de trabalho próximo ao mercado de software, com divisão de responsabilidades, integração de código e versionamento utilizando Git e GitHub.

🎯 Objetivo

Aplicar conceitos teóricos da Programação Orientada a Objetos em um projeto prático, reforçando também habilidades de colaboração em equipe, organização de código e boas práticas de desenvolvimento.

🧠 Conceitos Aplicados

O sistema foi estruturado com base nos principais pilares da POO:

Encapsulamento: proteção e controle de acesso aos dados;
Abstração: modelagem de entidades do sistema;
Herança: reutilização e extensão de comportamentos;
Polimorfismo: flexibilidade na implementação de métodos.
👥 Desenvolvimento em Equipe

O projeto foi desenvolvido em colaboração com outros alunos, promovendo práticas de desenvolvimento cooperativo, como:

Divisão de tarefas entre membros;
Integração de partes do sistema;
Uso de versionamento com Git;
Organização conjunta da estrutura do projeto.
⚙️ Tecnologias e Ferramentas
Linguagem de Programação Orientada a Objetos (conforme implementação do projeto)
Git & GitHub para controle de versão
IDEs de desenvolvimento utilizadas pela equipe
🚀 Resultados e Aprendizados

Este projeto contribuiu para o fortalecimento de competências técnicas e comportamentais, incluindo:

Aplicação prática de POO em um sistema funcional;
Experiência em desenvolvimento colaborativo;
Uso de ferramentas de versionamento em equipe;
Organização e estruturação de projetos acadêmicos;
Consolidação de boas práticas de programação.
📌 Contexto Acadêmico

Projeto desenvolvido como atividade avaliativa da disciplina de Programação Orientada a Objetos.
